# 💳 CreditMind AI: Smart Customer Behavior Prediction System

CreditMind AI is a machine learning powered financial intelligence platform designed to analyze customer behavior and support smarter banking decisions.

The system combines regression and classification techniques to solve two real-world banking problems:

- 💰 Predicting optimal customer credit limits based on financial behavior and profile data.
- 📉 Identifying customers at risk of leaving the bank using churn prediction.

The project implements multiple machine learning models including Linear Regression, Ridge Regression (L2 Regularization), Lasso Regression (L1 Regularization), and Logistic Regression. Model performance is evaluated using industry-standard metrics such as R² Score, RMSE, MAE, Accuracy, Precision, Recall, F1 Score, and ROC-AUC.

A modern interactive Streamlit dashboard provides real-time predictions with a user-friendly fintech-inspired interface.
